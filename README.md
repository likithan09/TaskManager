рџ”„ Algorithm

User Registration

1. User enters email and password.
2. System validates input.
3. Firebase creates user account.
4. Success message displayed.
5. Redirect to Login Page.

User Login

1. User enters credentials.
2. Firebase verifies credentials.
3. If valid, user is redirected to Dashboard.
4. If invalid, error message displayed.

Task Management

1. User enters task details.
2. Task is stored in Local Storage.
3. Task is displayed on Dashboard.
4. User can Edit/Delete/Complete tasks.
5. Statistics are updated automatically.
